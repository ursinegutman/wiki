# Installation Guide

## Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

## Steps

1. Clone the repository
2. Install dependencies: `pip install markdown pystache python-frontmatter`
3. Build: `python scripts/markdown2html.py`
